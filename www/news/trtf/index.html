Predictive Distribution Modelling Using Transformation Forests

New paper discussing a novel transformation tree algorithm leading to transformation forests, which allow to compute fully parametric yet very general predictive distributions.

An open-source implementation of transformation trees and transformation forests is available in the R add-on package trtf.
